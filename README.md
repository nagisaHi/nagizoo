# 柊渚の思考動物園 🐾

考えごとという生き物を放し飼いにしている、柊渚の個人サイト（試作）。

- **表**: 思考動物園 — 飼育舎（未来舎・宇宙舎・文明舎・人間舎・夜行館）、今週の一頭（哲学語録）、観察スタンプ帳、渚観察日誌
- **裏**: Still life — 人間の静物画。論考・書架・目録・水底ゼミ・Dive（入口はフッターの 🌘）
- **管理室**: フッターの ⚙ から（デモ）。投稿・観察日誌・園内マップ画像の差し替えができます

## 見かた

GitHub Pages で公開している場合はそのURLへ。ローカルなら `index.html` をブラウザで開くだけです。

## 園内マップの絵地図

リポジトリ直下に `map.jpg`（絵地図画像）を置くと、全訪問者にその絵地図が園内マップとして表示されます。無い場合は内蔵のSVG地図が出ます。

## 注意（試作につき）

- 投稿・スタンプ・購読状態などはすべて **閲覧者のブラウザ（localStorage）に保存** されます。サーバーはありません
- 管理室のパスワードはデモです（本物の認証ではありません）。実運用時は WordPress / microCMS 等に載せ替える前提です
